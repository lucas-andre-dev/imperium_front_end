# Imperium Frontend

O **Imperium Frontend** é a interface web do sistema Imperium.  
Desenvolvido em **HTML, CSS e JavaScript**, o projeto oferece uma experiência moderna e responsiva para usuários e administradores, permitindo a criação de chamados de suporte, envio de sugestões e controle administrativo.

---

## Tecnologias utilizadas
- HTML5  
- CSS3 (Bootstrap 5)  
- JavaScript (Fetch API)  
- Integração com API REST (Spring Boot)

---

## Funcionalidades principais
- Criação e gerenciamento de chamados de suporte de TI.  
- Envio e acompanhamento de sugestões de melhoria.  
- Login e controle de usuários.  
- Painel administrativo para gerenciamento de usuários, estoque e sugestões.  
- Comunicação direta com a API REST do backend.
